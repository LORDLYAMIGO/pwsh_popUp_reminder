Popup Reminder Script (PowerShell)

This script pops up a Windows message box at a fixed time interval to remind you to get back to work. It’s simple, direct, and does exactly what you tell it to — nothing more.

What It Does

Shows a popup alert with your custom message

Repeats after a set number of minutes

Runs indefinitely until you close the PowerShell window

If you close the terminal, the reminders stop. Don’t overthink it.
